Pasos para la ejecución:

1. Descargar "MusicShare.zip" de la última release en la sección de Releases (Menú de la derecha)
2. Guardar el ejecutable junto a la carpeta locals
3. Ejecutar el programa y seguir las instrucciones que se indican:
    * Autorizar a la aplicación en Spotify
    * Autorizar a la aplicación en Youtube (simplemente seguir los pasos que indica Youtube)
4. Seleccionar la opción introduciendo el número y ENTER
